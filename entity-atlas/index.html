Entity Atlas

One map of every platform object, its data, its events, and what automation can reach. Filters only change what is shown; the dataset underneath always keeps every relationship.

Read the map

Every circle is a platform object, coloured by object type and sized by how connected it is. Shaded areas are domain halls. Lines are relationships; their colour tells you the kind: blue ownership, grey reference, amber dependency, teal association.

Find anything

Press / and type. Search covers names, fields, events, modules, roles, and purpose text, then jumps straight to the entity, revealing it if a filter was hiding it.

Focus

Double-click an entity, or use Focus in the inspector, to see just its neighbourhood. Step the depth with 1 2 3. Esc returns to the full map.

Trace a path

Path mode finds the shortest relationship chain between any two entities: how Registration reaches Lead Record, for instance. Pick the start, pick the end, read the steps in the inspector.

Table for lists

The Table view shows the same filtered set as sortable rows: links, events, lifecycle, and automation surface per entity. Good for review, exports, and anyone who prefers rows to maps.
